# 🐍 Python Quest: Zero to Hero

An interactive, terminal-based learning console that teaches Python through a narrative "Hacker Quest."

## 🌟 Features
- **20 Structured Lessons**: From variables to Object-Oriented Programming.
- **Interactive Coach**: Step-by-step guidance through Concept, Syntax, and Challenge.
- **Cyber-Terminal UI**: High-tech dashboard with real-time progress bars.
- **Safe Sandbox**: Experiment with code freely in a crash-proof environment.
- **Persistence**: Remembers your name, level, and completed sagas.

## 🚀 How to Run
1. Ensure you have **Python 3.x** installed.
2. Download `study_buddy.py`.
3. Run it in your terminal:
   ```bash
   python study_buddy.py
   ```

## 🛠️ Commands
- **RUN**: Execute your code.
- **UNDO**: Remove the last line.
- **CLEAR**: Start fresh in the editor.
- **EXIT**: Return to the main menu.
- **RESET**: Wipe all progress and start a new quest.
